## SEO Checklist

### Before Publishing:

**Keyword Optimization:**
- [ ] Target keyword in title (front-loaded)
- [ ] Target keyword in first 100 words
- [ ] Target keyword in 2-3 H2 headings
- [ ] Target keyword in conclusion
- [ ] LSI keywords throughout (3-5x)

**Meta Information:**
- [ ] Meta description (150-160 chars)
- [ ] Includes target keyword
- [ ] Has clear CTA

**Content Quality:**
- [ ] Word count: 1,500-3,000
- [ ] Readability score: >60 (Hemingway)
- [ ] Short paragraphs (2-4 sentences)
- [ ] Bullet points and numbered lists
- [ ] Clear H2/H3 hierarchy

**Media:**
- [ ] Featured image (1200×675px, 16:9)
- [ ] Alt text for all images
- [ ] Images optimized (<200KB)
- [ ] 3-7 images total

**Links:**
- [ ] 3-5 internal links
- [ ] 2-3 external authoritative links
- [ ] All links working

**Technical:**
- [ ] Schema markup added
- [ ] Canonical URL set
- [ ] Mobile-friendly formatting
- [ ] Table of contents (if >1,500 words)

**Promotion:**
- [ ] Social media posts prepared
- [ ] Email newsletter drafted
- [ ] LinkedIn article scheduled
- [ ] Reddit posts planned

## Image Specifications

### Featured Image:
- **Size:** 1200 × 675 pixels
- **Ratio:** 16:9
- **Format:** WebP
- **File Size:** < 200KB
- **Content:** Relevant to topic, brand colors

### Inline Images:
- **Width:** 800-1200px
- **Format:** WebP or PNG
- **Alt Text:** Descriptive, includes keyword if natural
